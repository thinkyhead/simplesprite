SS_CustomGadget

Superclass: SS_Gadget

There are times when you need a special gadget but you don't want to make an entire subclass. The SS_CustomGadget augments the vanilla SS_Gadget class by adding a render handler pointer. SimpleSprite sets up the render context and does the event dispatching.

Code Example
  SS_CustomGadget *custom;

  custom = new SS_CustomGadget(
      myGUI,                        // An existing GUI
      150, 150,                     // 150 x 150 area
      myRenderProc                  // My own renderer
    );

  custom->SetHandler(SS_EVENT_CLICK, myEventProc);  // My event handler
  custom->Move(10, 10);                             // Offset in the GUI
