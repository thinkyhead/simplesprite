HandleMessage

void HandleMessage(const SS_Message &message)

void HandleMessage(
       Uint32 mclass,
       Uint32 mcode,
       int    mval=0,
       void   *data=NULL
       )

There are two forms of HandleMessage. The first form is the message handler itself. You should override this method in your derived classes. The second form takes the arguments to fill in a new SS_Message and then calls the message handler. It is intended for use as an on-the-fly message sender.

Ignore

void Ignore(SS_Broadcaster *sender)

Ignore tells the listener to stop listening to one of its broadcasters. (It tells the broadcaster to stop sending messages to this listener.)

top

IgnoreAll

void IgnoreAll()

IgnoreAll tells the listener to stop listening to all its broadcasters. (It tells all the broadcasters to stop sending messages to this listener.)

ListenTo

void ListenTo(SS_Broadcaster *sender)

ListenTo tells the listener to listen to a broadcaster. (It adds this listener to the broadcaster's list of listeners.)
