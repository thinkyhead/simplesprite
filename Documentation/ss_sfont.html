SS_SFont.h

SS_SFont

The format is popular among SDL and Linux game developers for its simplicity and portability. Making an SFont file is pretty simple. SimpleSprite doesn't use the official SFont library, but uses the SFont format as the basis for the SS_SFont class. You can use any SDL-compatible image format as an SFont (PNG is best).

floatdescenderThe height of the descender

Constructors

SS_SFont(char *filename, float desc)

The constructor requires the path to an image file containing a valid SFont image and the height of the descender. During construction the specified image is loaded into memory, converted into gl textures and display lists - one of each per character.
