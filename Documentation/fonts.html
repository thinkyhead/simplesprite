Fonts & Strings

SimpleSprite uses a format called "SFont" for its font files. An SFont file is an ordinary image file with 96 ASCII characters rendered in the exact size and color that they will appear in your game.

Space Markers consist of one or more pixels of solid color that mark the space between character images. The official SFont format requires the markers to be pure magenta (0xFF00FF) but in SimpleSprite any color will work. The character graphic extends from line below the marker to the next marker line (or the bottom of the image). The marker line isn't included as part of the loaded font, so the non-marker pixels in the marker line can also contain any color.

The official SFont format requires all characters to be positioned on a single line, which makes for a very wide image. SimpleSprite allows you to use as many lines as you like. If you use the multi-line format you must make sure that every line of characters is the same height. (i.e., The marker lines must be equally-spaced.) Regardless of how many lines are used, there must be a marker after the last character on every line.
