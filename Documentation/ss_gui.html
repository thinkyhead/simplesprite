SS_GUI.h

SS_GUI

Superclass: SS_Layer

The SS_GUI is a special type of SS_Layer. It contains SS_Gadgets, which have interactive abilities through the SDL event dispatcher. Gadgets automatically receive events pertaining to mouse movement, mouse clicks, drags, and keypresses which are then handled by your event procedures. Unhandled events are passed back to the SS_GUI, which can then pass it on to other GUIs and eventually the SS_World.

Code Example:
  SS_World *w = new SS_World();
  SS_GUI *theGUI = w->NewGUI();

  theGUI->SetFont("font/optima18.png", 4);

  cd("gui");

  theGUI->LoadSliderImages(
    "slider1.png",
    "thumb1.png",
    "thumb2.png",
    "thumb3.png"
  );

  theGUI->LoadCheckboxImages(
    "checkbox1_off.png",
    "checkbox1_on.png",
    "checkbox1_off_over.png",
    "checkbox1_on_over.png"
  );

  cd("..");
