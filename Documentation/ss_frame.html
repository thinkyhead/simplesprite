SS_Frame.h

SS_Frame

SuperClass: SS_RefCounter

The SS_Frame class represents the bitmap data which SS_Sprite utilizes for its frames. When you create a frame you must supply either the pathname of an image or an existing SDL_Surface. You can also pass flags indicating whether the frame should generate a collision mask and whether it should retain its surface data.

MakeCollisionMask

void MakeCollisionMask()
void MakeCollisionMask(Uint8 threshold)

Create a collision mask for the frame. The frame's collision mask is a one-bit representation of the frame which is used to precisely detect collisions. This gives the highest level of precision possible but bit-level collision detection can be expensive. See the notes on collision-detection elsewhere in this doc.
