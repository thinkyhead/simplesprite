Base Classes

The classes described in this section provide support for most of the classes in this framework. You don't need to use these classes directly; however you will need to be familiar with their methods to fully utilize most of the other classes.

SS_RefCounter
This class provides "reference counting" to help prevent premature deletion and memory leaks.
SS_ListNode<T> • SS_LinkedList<T> • SS_ObjectList<T> • SS_Iterator<T>
Linked Lists are essential. These classes provide the basic functionality for linked lists of simple data types, aggregate data types, and object pointers. They are template classes, so SimpleSprite uses typedefs to declare the specific data types it needs.
