SS_Messages.h

SS_Broadcaster

Subclasses: SS_Gadget | SS_Layer | SS_LayerItem | SS_World

Any object that wants to send messages to another object should inherit from SS_Broadcaster. Then when some interesting event occurs within the object it can broadcast a message.
