AlignToPoint

void AlignToPoint(float x, float y, stringAlign position=(SA_LEFT|SA_CENTERV))

Move the string to the given position within its container and optionally set its alignment. Alignments for this method and others that take a position are:

    Horizontal:   SA_LEFT     SA_RIGHT    SA_CENTERH
      Vertical:   SA_ABOVE    SA_BELOW    SA_BASELINE    SA_CENTERV
Center h and v:   SA_CENTER
