SS_Vectors.h

SS_VectorFrame

SuperClass: SS_RefCounter

The SS_VectorFrame class embodies the vectors and instructions for drawing geometrical forms. A single vector-frame can contain an unlimited number of lines, boxes, circles, and other shapes, each with its own colors, translucency, line weight, and other properties.
