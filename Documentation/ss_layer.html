SS_Layer.h

SS_Layer

Superclass: SS_ItemList

Subclasses: SS_GUI | SS_TextLayer | SS_TileLayer

The SS_Layer class allows you to group sprites and other game objects into independently controlled units. You can change the order of layers, and you can show, hide, pause, and disable them. Layers can be world- synchronized to scroll and zoom in alignment with the world. Or they can be set to a fixed position for HUD displays and interface objects. (Although called "fixed," they actually have x/y offsets of their own.)

Constructors

SS_Layer(SS_World *world, Uint32 flags=SS_NONE)

When you create a new empty layer you can optionally add it to an existing World and/or set the layer flags. The available flags are:

SS_NONEA normal layer
SS_NOSCROLLDon't apply world-scrolling
SS_NOZOOMDon't apply world-zooming
