SS_Messages.h

SS_Message

When a message is sent from a Broadcaster to a Listener the HandleMessage method receives an SS_Message reference. The receiver then uses the contents of the SS_Message to determine the course of action. An SS_Message is composed of a message class, a message code, an integer message value, and an optional pointer to sender-defined data.

Member Data

Constructors

SS_Message()

You can construct an SS_Message with an empty message or pre-populated. Note that the SS_Listener::HandleMessage method can take an SS_Message& argument. This makes it possible to create, send, and delete a message all in one line: 

        HandleMessage(SS_Message('FUN ','MESG',1975))
