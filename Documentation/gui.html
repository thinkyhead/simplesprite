User Interface

A Gadget is a graphical object that players can interact with. A GUI is a graphical user interface made with one or more Gadgets. SimpleSprite implements several types of gadget already, such as buttons and checkboxes. Gadgets respond to various kinds of events.

Events

When an event occurs the pertinent gadgets are handed a pointer to an SS_Event structure that describes what happened. Your gadget's handler uses the data in the event structure to do something useful. 

typedef struct {
  ssEventCode  type;                    // The event code
  float        x, y;                    // Mouse pos in gadget coords
  float        xrel, yrel;              // Mouse motion
  float        xgui, ygui;              // Mouse pos in gui coords
  float        xglob, yglob;            // Mouse pos in screen coords
  SDLKey       key;                     // The pressed key
  SDLMod       mod;                     // Key modifiers
} SS_Event;

Events are automatically dispatched to gadgets as they occur. The supplied gadgets handle certain events themselves. For example, a Slider updates the thumb position as you click and drag on it. For your own special behaviors you need only set event handlers for the events that matter. The following events are available:

SS_EVENT_ENTERThe gadget was entered
SS_EVENT_MOVEThe mouse moved within the gadget
SS_EVENT_EXITThe mouse exited the gadget
SS_EVENT_CLICKThe gadget was clicked
SS_EVENT_DRAGThe gadget was dragged
SS_EVENT_UNCLICKThe gadget was unclicked
SS_EVENT_KEYDOWNA key was pressed
SS_EVENT_KEYUPA key was released
SS_EVENT_GAIN_FOCUSThe gadget gained focus
SS_EVENT_LOSE_FOCUSThe gadget lost focus
SS_EVENT_CHANGEThe value changed
SS_EVENT_IDLENothing happened
