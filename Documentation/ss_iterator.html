SS_Iterator<T>

About this class...

Member Data

Constructors

SS_Iterator(SS_LinkedList<T>* p_list=NULL, SS_ListNode<T>* p_node=NULL)

You will never need to call the constructor for SS_Iterator<T> directly. Instead you'll get an iterator from the list you want to iterate by calling its GetIterator() method. The code sample below illustrates the use of an iterator to process an entire linked list.

Code Example
  typedef TObjectList<Doorknob*> KnobIterator;   // Easy to make!

  Doorknob *knob;
  KnobIterator itr = knobList.GetIterator();     // Get the iterator
  while (knob = itr.NextItem())                  // Each piece of data
    knob->TurnClockwise();
